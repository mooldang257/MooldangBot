# 📜 [성전]: MooldangBot 변수 치트 시트 (Variable Cheat Sheet)

오시리스 함대의 동적 명령어 파싱 엔진에서 사용되는 모든 표준 변수 목록입니다. 모든 변수는 반드시 **`$(변수명)`** 형식을 준수해야 하며, 대소문자를 구분하지 않으나 가급적 명시된 표기법을 권장합니다.

---

## 👥 시청자 및 스트리머 관련 (User Context)

| 변수명 | 설명 | 출력 예시 |
| :--- | :--- | :--- |
| `$(닉네임)` | 현재 명령어를 입력한 시청자의 닉네임 | `mooldang257` |
| `$(스트리머)` | 현재 방송 중인 스트리머의 닉네임 | `mooldang` |
| `$(UID)` | 시청자의 고유 식별자 (Chzzk UID) | `a1b2c3d4...` |

## 💰 경제 및 포인트 (Economy)

| 변수명 | 설명 | 출력 예시 |
| :--- | :--- | :--- |
| `$(포인트)` | 현재 시청자가 보유한 포인트 | `1,500` |
| `$(순위)` | 함대 내 전체 포인트 순위 | `12` |
| `$(누적포인트)` | 시청자가 지금까지 획득한 총 포인트 | `50,000` |

## 🕹️ 방송 및 명령어 (System & Stream)

| 변수명 | 설명 | 출력 예시 |
| :--- | :--- | :--- |
| `$(현재시각)` | 현재 시스템 시각 (HH:mm:ss) | `13:05:22` |
| `$(날짜)` | 오늘 날짜 (yyyy-MM-dd) | `2026-04-04` |
| `$(방송제목)` | 현재 치지직 방송 제목 | `[물멍] 오시리스 기함 정비 중` |
| `$(카테고리)` | 현재 설정된 방송 카테고리 | `Software Development` |
| `$(업타임)` | 방송 시작 후 경과 시간 | `02:45:10` |

## 🎲 룰렛 및 확률 (Roulette)

| 변수명 | 설명 | 출력 예시 |
| :--- | :--- | :--- |
| `$(룰렛결과)` | 방금 실행된 룰렛 당첨 항목명 | `치킨 세트` |
| `$(룰렛이름)` | 실행된 룰렛의 명칭 | `야식 룰렛` |

---

## ⚠️ 작성 가이드라인

1. **중첩 금지**: `$(foo $(bar))`와 같은 중첩된 변수 호출은 지원하지 않습니다.
2. **이스케이프**: 문구 내에 `$(...)` 문자열을 그대로 노출하고 싶다면 `$$(...)`로 작성하십시오.
3. **미정의 변수**: 존재하지 않는 변수명을 입력하면 공백이나 에러 대신 입력값 그대로(`$(Unknown)`)가 출력됩니다.

> [!TIP]
> **Regex 공명**: 본 변수들은 `DynamicQueryEngine` 내의 `\$\((?<varName>.*?)\)` 패턴에 의해 공명(Parsing)됩니다.
