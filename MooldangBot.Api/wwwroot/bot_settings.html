🤖 전용 봇 및 API 설정

스트리머 본인만의 치지직 Open API 앱 정보를 사용하여 전용 봇을 운영합니다.

⚠️ 주의사항
본 설정을 완료하려면 치지직 개발자 센터에서 직접 앱을 생성해야 합니다.
설정된 정보가 정확하지 않으면 봇이 작동하지 않거나 로그인이 실패할 수 있습니다.

🔑 API 앱 정보 설정

치지직 개발자 센터 앱 설정의 'Redirect URL' 항목에 위 주소를 정확히 입력해 주세요.

🤖 봇 계정 연동

위 API 설정을 기반으로 봇 기능을 수행할 계정을 연동합니다.

현재 연동된 봇: -
로그인 전 확인!
아래 버튼을 누르면 현재 브라우저에 로그인된 치지직 계정이 봇으로 연동됩니다.
본인 계정을 봇으로 쓰시려면 그대로 진행하시고, 별도의 봇 계정을 쓰시려면 해당 계정으로 로그인 후 버튼을 눌러주세요.
